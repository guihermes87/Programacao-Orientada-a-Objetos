Manipulação de arquivos existentes dentro do java.io

FileWriter: Escreve caracteres em um arquivo. Ele é usado para criar arquivos de texto e escrever dados neles. Ele pode ser usado para escrever caracteres, strings e arrays de caracteres.
FileReader: Lê caracteres de um arquivo. Ele é usado para ler arquivos de texto e obter os dados contidos neles. Ele pode ser usado para ler caracteres, strings e arrays de caracteres.
BufferedWriter: Escreve caracteres em um arquivo de forma eficiente, usando um buffer. Ele é usado para melhorar o desempenho ao escrever grandes quantidades de dados em um arquivo. Ele pode ser usado para escrever caracteres, strings e arrays de caracteres.
BufferedReader: Grandes quantidades de arquivos. Lê caracteres de um arquivo de forma eficiente, usando um buffer. Ele é usado para melhorar o desempenho ao ler grandes quantidades de dados de um arquivo. Ele pode ser usado para ler caracteres, strings e arrays de caracteres.
