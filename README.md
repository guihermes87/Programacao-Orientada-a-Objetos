# Programação Orientada a Objetos

Repositório com exercícios e códigos de aula da disciplina de **Programação Orientada a Objetos** em Java.

## Estrutura

Os exercícios estão organizados por tema, em ordem cronológica das aulas:

| Pasta | Conteúdo |
|-------|----------|
| `01-fundamentos-java/` | Exercícios iniciais — primeiras classes, atributos e métodos |
| `02-classes-e-objetos/` | Aula de 17/03 — criação de classes e instâncias |
| `03-tratamento-de-erros/` | Aula de 24/03 — try/catch e exceções |
| `04-exercicios-31-03/` | Aula de 31/03 — exercícios variados (Filme, Carro, Time, etc.) |
| `05-encapsulamento/` | Aula de 07/04 — getters/setters e modificadores de acesso |
| `06-revisao/` | Aula de 14/04 — exercícios de revisão |
| `07-heranca/` | Aula de 28/04 — herança, super, cascateamento |
| `08-sobrecarga-e-sobrescrita/` | Aula de 05/05 — overload e override |
| `09-arraylist/` | Coleções com ArrayList |
| `10-aula-12-05/` | Aula de 12/05 — Pessoa, Endereço, Produto |
| `13-arquivos/` | Manipulação de arquivos — leitura/escrita de texto (cadastro de produtos) |
| `prova-pratica/` | Prova prática |

## Como executar

Cada pasta contém um ou mais conjuntos de classes Java. Para compilar e executar:

```bash
cd 02-classes-e-objetos/ExerciciosAula1703
javac *.java
java Principal
```
